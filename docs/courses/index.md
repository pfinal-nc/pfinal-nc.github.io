## 🚀 核心课程

### 🔥 热门课程

#### 1. [Go 后端工程师成长路线](/courses/golang-backend/)
**难度：** 🟢 入门 → 🔴 高级 | **预计时长：** 8-12 周

```
📦 5 大模块 · 20+ 课时 · 完整实战项目
```

**课程大纲：**
- **模块 1：Go 核心进阶** - 并发编程、内存管理、GC 调优
- **模块 2：Web 框架实战** - Gin/Echo、中间件、API 设计
- **模块 3：数据库设计与优化** - MySQL/PostgreSQL、Redis、ClickHouse
- **模块 4：微服务架构** - 服务拆分、通信、治理
- **模块 5：可观测性体系** - 日志、指标、Trace、告警

**实战项目：** 从零搭建企业级博客 API 系统

[进入课程 →](/courses/golang-backend/)

## 💡 学习建议

### 1. 选择适合你的起点
- **零基础**：从 Go/Python 语言基础开始
- **有經驗**：直接学习进阶课程或专题

### 2. 按顺序学习
- 课程内部模块建议按顺序学习
- 前后模块有知识依赖关系

### 3. 动手实践
- 每个课时都有代码示例
- 完成课后练习和实战项目

### 4. 持续复习
- 技术需要反复练习
- 建议做笔记和总结
