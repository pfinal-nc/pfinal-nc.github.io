### 代码块示例

以下是一个简单的Go语言代码示例，用于验证代码块的语言标签是否正确：

```go
package main

import "fmt"

func main() {
    // 打印欢迎信息
    fmt.Println("Hello, Go!")
    
    // 计算斐波那契数列
    n := 10
    fmt.Printf("Fibonacci(%d) = %d\n", n, fibonacci(n))
}

func fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    return fibonacci(n-1) + fibonacci(n-2)
}
```
