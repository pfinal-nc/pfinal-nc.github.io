    gopsutil 介绍

    gopsutil 是 psutil（一个常用于 Python 的系统和进程监控库）的 Go 实现版本。通过它，我们能够跨平台地获取系统相关的信息，且其 API 设计非常简洁易用，支持多种系统监控需求。

    PS: gopsutil 将不同的功能划分到不同的子包中：

    • cpu: CPU 相关
    • disk: 磁盘相关
    • mem: 内存相关
    • host: 主机相关
    • 其他还有 docker, net, process, winservices 等

    内存信息

    通过 gopsutil/mem 可以获取系统的内存使用情况
