  • Elasticsearch1.7

使用方式

  1. 安装好 docker

  2. 安装好 docker-composer

  3. clone源码

  1. 复制 .env.example 为 .env,并且进行配置

ES_UID=$(id -u):$(id -g) docker-composer up -d

docker ps

http://localhost:9000 就可以看到 portainer.io 界面来管理容器
